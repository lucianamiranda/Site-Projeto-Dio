Ferramentas utilizadas

(Voltar)

Atualmente no ambiente de desenvolvimento existem muitas ferramentas estruturadas que são disponibilizadas para o desenvolver em seu dia a dia.No curso é apresentado a ferramenta Visual Studio Code, um editor de código sendo possível instalar em seu computador de forma gratuita e objetiva.Essa ferramenta é capaz instalar e habilitar pluggins que serão necessários para facilitar no processo de codificação.

Inspetor de elementos

(Voltar)

Encontrado no navegador de uso , o inspetor de elementos é um recurso de vizualização e modificação temporária do código-fonte de qualquer página web.

Estrutura básica do HTML

(Voltar)

A estrututa básica do HTML é formada por três elementos:HTML,head e body. O código é aberto com o comando html e fechado com o mesmo , logo após é colocado o comando head em que são carregadas as informações antes de chegar no usuário final. Finalizando com a chave body , nele é exibido todo conteúdo para o usuário

Tags

(voltar)

As tags Strong,i,input e img são capazes de deixar o site um pouco mais estilizado. A tag Strong é responsável por deixar o elemento em negrito,o i no formato italico,Input juntamentamente com o atributo "type" adiciona campo para adicionar informações de tipos variados como number,texto e cores.

Atributos das tags

(voltar)

Os atributos juntamente com as tags possibilitam controlar o comportamento do elemento dentro do código.A palavra reservada id identifica o elemento, style vindo do Css é capaz de personalizar o texto em sua cor,class é reservada para tornar elementos no código padrão,organizado em um único estilo. o atributo anteriomente citado type torna específico da tag imput em que podemos adicionar campos formatados em texto,número e cores.Por fim , em conjuto com a tag imga palavra reservada scr identifica o caminho da imagem colocada podendo modificar sua largura com o termo width

Textos

(voltar)

A estrutura para adicionarmos textos é possível com as tags de h1 até h6 que vão aumentando conforme a numeração , o termo blockquote exibe o texto em citação tendo mais o sup que adiciona um asterístico para personalizar.No mais ,a tag p formata o texto em paragráfos fazendo com que a estrutura fique organizada.

Lista ordenada

(voltar)

A lista ordenada é aberta com o comando ol em que lista os termos numericamente ,juntamente com o termo li que ordena as informações.

Lista não ordenada

(voltar)

A lista não ordenada possui o termo de entrada ul juntamewnte com li em que as informações são listadas com marcações.

Links

(voltar)

Através do link podemos acessar outros sites, sendo aberto com o termo a seguindo de um atributo href em que se coloca o caminho da informação/site desejado.No mais sendo possível adicionar o atributo target indicando a abertura na mesma aba ou em outra posterior e title podendo exibir mensagem quando passado o mouse sobre o link.
